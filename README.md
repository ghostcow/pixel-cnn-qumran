
# pixel-cnn++

This is a fork of OpenAI's Implementation of [PixelCNN++](https://openreview.net/pdf?id=BJrFC6ceg), as described in the following paper:

**PixelCNN++: A PixelCNN Implementation with Discretized Logistic Mixture Likelihood and Other Modifications**, by
Tim Salimans, Andrej Karpathy, Xi Chen, Diederik P. Kingma, and Yaroslav Bulatov.

The fork runs on a dataset of letters from the ancient Dead Sea Scrolls with some modifications.
